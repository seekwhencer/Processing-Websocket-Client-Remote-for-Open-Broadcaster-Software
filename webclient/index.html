Facebook Object Comments

How to use Facebook Comments from an Object (Post, Photo etc.)

  • Download the PHP-Sources, place it on a Webserver who is accessible by facebook - and make it runable, set rights to write files
  • Go to Facebook, create an App and copy your Access Token (User Token from App)
  • Edit the "conf.php" of the PHP Sources: Access Token, App Secret, App Id
  • Call the "token.php" in your Browser to generate a long lived (60 Days) Access Token. The long lived Token where stored as "log/token".
  • The long lived Token Activation invokes automatically after token.php is complete. You can call it manually: "longtoken.php"
  • Now you can make api calls to facebook with the 60-days Token.

How to add Comments Layer in Open Broadcaster Software?

  • Create a new global source with the Plugin CLR Browser Plugin
  • Use the "http://myhost/my/folder/comments.html" (from this folder here) as source.
  • Add a new Source by using the global Source
  • The comments.html calls now every 10th Second
Facebook Object Id
